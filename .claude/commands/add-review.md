# 리뷰 추가/수정

메인 화면의 네이버 리뷰 섹션에 새 리뷰를 추가하거나 기존 리뷰를 수정합니다.

## 담당 파일

`src/locales/ko.ts`의 `review.reviews` 배열 (및 다른 언어 파일 동일 위치)

## 리뷰 항목 구조

```
{ name: '닉네임', tag: '예약 후 이용 · 대기 10분 이내', text: '리뷰 내용' }
```

- `name`: 네이버 닉네임 (예: `sop***215`)
- `tag`: 방문 유형 (예: `예약 후 이용 · 대기 10분 이내` 또는 `예약 없이 이용 · 바로 입장`)
- `text`: 실제 리뷰 텍스트

## 작업 방법

### 리뷰 추가

1. 클라이언트에게 닉네임, 방문 유형, 리뷰 내용을 확인한다.
2. `src/locales/ko.ts`의 `review.reviews` 배열 끝에 추가한다.
3. 나머지 4개 언어 파일에도 동일하게 추가한다.
   - 한국어 리뷰는 번역하지 않고 **원문 그대로** 유지한다. (실제 고객 리뷰이므로)

### 리뷰 삭제

1. 삭제할 리뷰의 닉네임 또는 내용을 확인한다.
2. 5개 언어 파일 모두에서 해당 항목을 삭제한다.

### 리뷰 수정

1. 수정할 리뷰를 닉네임으로 찾는다.
2. 5개 언어 파일 모두에서 동일하게 수정한다.

## 주의사항

- 리뷰는 실제 고객 후기이므로 내용을 임의로 변경하지 않는다.
- 현재 리뷰 수: 4개 (너무 많으면 가독성이 떨어지므로 6~8개 이하 권장)
