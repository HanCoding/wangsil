# 텍스트/문구 수정

웹사이트의 텍스트, 문구, 설명글을 수정합니다.

## 담당 파일

모든 텍스트는 아래 5개 언어 파일에서 관리합니다:
- `src/locales/ko.ts` — 한국어 (원본)
- `src/locales/en.ts` — 영어
- `src/locales/ja.ts` — 일본어
- `src/locales/vi.ts` — 베트남어
- `src/locales/zh.ts` — 중국어

## 페이지별 주요 섹션

| 위치 | 섹션 키 |
|------|---------|
| 메인 히어로 배너 | `hero` |
| 메인 시술 소개 카드 | `treatment.items` |
| 메인 리얼셀카 섹션 | `selfie` |
| 메인 리뷰 섹션 | `review` |
| 푸터 | `footer` |
| 이벤트 팝업 | `popup` |
| 병원소개 페이지 | `about` |
| 눈성형 페이지 | `eye` |
| 코성형 페이지 | `nose` |
| 안면거상 페이지 | `facelift` |
| 쁘띠 시술 페이지 | `petti` |
| 레이저 페이지 | `laser` |
| 오시는길 페이지 | `location` |

## 수정 방법

1. 클라이언트의 요청에서 **어느 페이지, 어느 위치**의 텍스트인지 파악한다.
   - 불명확하면 후보를 제시하고 확인을 요청한다.
2. `src/locales/ko.ts`에서 해당 텍스트를 찾아 수정한다.
3. **5개 언어 파일 모두** 동일한 위치를 번역하여 수정한다.
   - 한국어가 원본이며, 나머지는 한국어 기준으로 적절히 번역한다.

## 주의사항

- `\n`은 줄바꿈이므로 문장 흐름에 맞게 유지한다.
- 의료 정보(수술 시간, 마취 방법 등)는 클라이언트가 제공한 값 그대로 입력한다. 임의로 변경하지 않는다.
- 수정 후 어떤 파일의 어떤 항목을 변경했는지 요약해서 알려준다.
